# GitHub MCP デモプロジェクト

このリポジトリは、GitHub MCPの機能をテストするためのデモプロジェクトです。

## 含まれるファイル

- `hello.js` - 簡単なJavaScriptの挨拶スクリプト
- `package.json` - Node.jsプロジェクト設定

## GitHub MCP で試す機能

1. ✅ リポジトリ作成
2. 🔄 ファイル作成・更新
3. 🔄 ブランチ作成
4. 🔄 プルリクエスト作成
5. 🔄 Issue管理

## 実行方法

```bash
node hello.js
```

これで「こんにちは、世界さん！」のメッセージが表示されます。

## セットアップ

```bash
npm install
npm start
```
